该项目为个人博客系统，普通用户功能主要包括发表文章，评论、点赞，可对发表过的文章及评论进行管理，用户之间可以互相访问对方主页，相互留言等。管理员可对普通用户进行权限管理，以及对系统文章库和评论库的管理。
